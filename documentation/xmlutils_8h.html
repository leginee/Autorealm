wxXmlNode* findNamedChild ( const wxString &  cname,
const wxXmlNode &  el 
)

Finds the first child of an xml element which has a given named.

Basically, this is a convenience routine which cycles through the children of a given XML element, and checks to see if any of them have a given name. If so, it returns the first one it finds.

Parameters:
cnameThe name of the child element to find
elThe wxXmlNode to be searched
Returns:
a pointer to the node found, or NULL if no node found

Constructs a new wxXmlNode and gives it data as a child element.

This routine is useful for creating an xml node of the form:

 <element>hello</element>

eldataThe data to be attached to the element ("hello" in the above example)
