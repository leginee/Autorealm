wxUint32 getRndNumber ( wxUint32  range)

An implementation of the minimal standard random number generator listed above.

This is a standardized random number generator for use across pretty well all platforms. At the time it was first written, the major concern was for the ability to work on even 16 bit systems. While this is no longer a concern, the random number generator still reflects some of these concerns. For further details about why to choose this pseudo-random number generator over any others, please refer to "The Communications of the ACM" October 1988, Vol. 31, Issue 10.
