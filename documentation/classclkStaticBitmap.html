Detailed Description

wxStaticBitmap does not pass mouse clicks up the event tree, but we need it to do so. This class, directly derived from it, does just that.

This is, mainly, a utility class. All it does is process a left single click or a left double click by passing it up to the parent's event handler. This is needed because wxStaticBitmap, the parent class, does not do this, and FillPattern does rely on it. In addition, it may be useful elsewhere.
